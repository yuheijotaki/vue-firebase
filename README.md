### 環境プレビュー&ビルド

#### ローカル起動

```
$ npm run serve
```

#### ビルド

```
$ npm run build
```

### Firebase

#### ログイン

```
$ firebase login
```

#### ローカルでデプロイ内容確認

```
$ firebase serve
```

#### デプロイ

```
$ firebase deploy
```
